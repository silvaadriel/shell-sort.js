      Shell Sort

      Criado por Donald Shell em 1959, publicado pela Universidade de Cincinnati, Shell sort é o mais eficiente algoritmo de classificação dentre os de complexidade quadrática. É um refinamento do método de inserção direta.

      Shell Sort vs Insertion Sort

      O algoritmo difere do método de inserção direta pelo fato de no lugar de considerar o array a ser ordenado como um único segmento, ele considera vários segmentos sendo aplicado o método de inserção direta em cada um deles. Basicamente o algoritmo passa várias vezes pela lista dividindo o grupo maior em menores. Nos grupos menores é aplicado o método da ordenação por inserção.

      Código em C

        #include <stdio.h>
  #include <stdlib.h>

  int shellSort(int *vetor, int tamanhoVetor) {
    int intervalo = tamanhoVetor / 2;
    int auxiliar, i, j;
    int trocas = 0;
    
    while (intervalo > 0) {
      for (i = intervalo; i < tamanhoVetor; i++) {
        auxiliar = vetor[i];
        j = i;
        
        while (j >= intervalo && auxiliar < vetor[j - intervalo]) {
          vetor[j] = vetor[j - intervalo];
          j = j - intervalo;
          trocas++;
        }
        vetor[j] = auxiliar;
      }
      intervalo = intervalo / 2;
    }
    return trocas;
  }

  int main() {
    int vetor[] = {9,8,7,6,5,4,3,2,1,0};
    int tamanhoVetor = sizeof(vetor) / sizeof(vetor[0]);
    int numeroTrocas;
    
    for (int i = 0; i < tamanhoVetor; i++) {
      i == tamanhoVetor - 1 ? printf(" %i", vetor[i]) : printf(" %i,", vetor[i]);
    }
    printf("\n");
    
    numeroTrocas = shellSort(vetor, tamanhoVetor);
    
    for (int i = 0; i < tamanhoVetor; i++) {
      i == tamanhoVetor - 1 ? printf(" %i", vetor[i]) : printf(" %i,", vetor[i]);
    }
    printf("\n\nNumero de trocas: %i", numeroTrocas);
    
    return 0;
  }

      Conclusão

      Vantagens:
      Shell Sort é uma ótima opção para arquivos de tamanho moderado.
      Sua implementação é simples e requer uma quantidade de código pequena.
      Desvantagens:
      O tempo de execução do algoritmo é sensível à ordem inicial do arquivo.
      O método não é estável.
